Projects

Fish Bowl

[Video]

Fish Bowl is a rhythm game based on the Kero Kero Bonito song of the same name.
Join Squish on a simple but challenging stroll through his home and enjoy the sights along the way!
*Prototype made in a weekend.

Karate Man 3D

[Download] [Play in Browser]

A recreation of the Karate Man minigame from the Rhythm Heaven series in Unity. Punch flying objects to the beat as part of your training!
Made this in a week as a base case to build a rhythm minigame framework.

Tempest Game Jam

Remixed the classic arcade game Tempest in C++, adding new features like a spin mechanic, a combo system, and custom maps.
Submission for Ubisoft Toronto NEXT Competition 2020.

Cat and Mouse A.I.

Developed artificial intelligence for a cat and mouse game where the AI mouse must collect all the cheese while avoiding the cats chasing it.
Applied techiniques including A*, minimax, and feature-based q-learning to ensure the mouse would win in a variety of scenarios.

Ray Tracer

2D and 3D ray tracer built in C, using linear algebra concepts and Phong illumination model to render complex scenes, also with the ability to use path tracing.
Implemented antialiasing, texture/normal mapping, refractive objects, multithreading, soft shadows, depth-of-field, and more.

Balloonatics

[Repository]

Developed an arena platformer using GameMaker:Studio set in a grotesque hellscape, where running in any direction will loop you back to where you began.
Use your balloon for gliding and blasting to survival!

Tractify

[Repository]

Powerful pipeline for generating tractograms on the Nipype framework, built for preprocessing neuroimaging data efficiently and reproducibly.
Combines top-of-the-line features from neuroscience and programming, including containerizing for supercomputing and new skullstripping method from niworkflows.
